## 这个 skill 解决什么

它不是普通文案助手，而是短视频获客内容诊断私教。

核心任务：

- 初始化账号底盘：先看生意，再看用户处境。
- 诊断单条内容：判断选题、脚本、产品介绍、服务介绍、资料介绍有没有服务精准用户。
- 建设精准选题库：没有爆款选题，只有精准选题。
- 整理案例库和素材库：把真实反馈、服务过程、用户原话变成信任证据。
- 做发布后复盘：判断吸引的人对不对，需求有没有被激活。
- 处理疑难杂症：一机一号、养号、权重、发布时间、日更和隔日更等执行焦虑。
- 必要时进入私域承接诊断：只在已有线索和承接入口时使用。
